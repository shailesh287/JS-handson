5. Shift and Unshift

shift() is a built-in JavaScript function that removes the first element from an array. The shift() function directly modifies the JavaScript array with which you are working. shift() returns the item you have removed from the array. The shift() function removes the item at index position 0 and shifts the values at future index numbers down by one.
