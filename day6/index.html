FUNCTIONS

Function declaration

            It is a block of code which you want to use multiple timestamp,
            so you define or declare it once, and then use it wherever you want as many times.

            function functionName() {
            	- - - code - - - 
            }
	
            functionName();                 (Invoking function)

Function expression

            A JavaScript function can also be defined using an expression.

             const z = function () {
                 console.log("inside function")
               };
               z();

            The function above is actually an Anonymous function (a function without a name).
            Functions stored in variables do not need function names. 
            They are always invoked using the variable name.

Arrow Function

            Arrow functions were introduced in ES6.
            const b = ()=> {
                — — — code — — —
            }
            b();

Function with "return" statement

            Whenever you are having a return statement in your function, 
            then at the time of invocation of the function, 
            do not invoke the function directly, instead of that, 
            assign the invocation of the function to a variable and the value which you 
            are returning from the function will be now stored in this new variable which you have created.

            function add(num1,num2){
                const result = num1 + num2;
               return result;
              }

            var cal = add(5,5);
            console.log(cal);

Constructor function

        In JavaScript, a constructor function is used to create objects.
           
         function Person () {                 (constructor function)
              this.name = 'John',
              this.age = 23
            }
            const person = new Person();
            console.log(person.name)

         function Person() is an object constructor function.
            To create an object from a constructor function, we use the new keyword.

Immediately invoked function


         (function (a,b){
                console.log(a+b);
            })(2,3);
            
         !function(){
                let a = "asdfgh";
                console.log(a);
            }();

            An IIFE (Immediately Invoked Function Expression) is a js function which runs as soon as it is defined.

CallBack Function

        A callback function is a function passed into another function as an argument.

         function yz(){
                console.log("hello i'm yz function");
            }
         function ac(newFunc){
                console.log("hello i'm ac function");
               newFunc();
            }
             ac(yz);

First class function

             Function which gets passed as an argument to a function.
             Function which can be returned from a function.
             Function which can be assigned to a variable as a value.

             const yz = function (){
                 console.log("hello i'm xyz function");
             }
             function ac(newFunc){
                 newFunc();
             }
             ac(yz);

Higher order function

             Function which gets passed as an argument to a function.
             Function which can be returned from a function.

             function yz(){
                 console.log("hello i'm xyz function");
             }
             function ac(newFunc){
                 newFunc();
             }
             ac(yz);
