charAt

If we want to find the characters of a string with the help of indexes we will use charAt() 

      let str1 = "askdasmsxsldfldkfa";
      let res = str1.charAt(9);
      console.log(`character at index 9 = ${res}`);

Split

converts string to an array, split method returns the new array, does not the change the original one. 

      let text = "ShaileshBind";
      var res3 = text.split("");
      console.log(`${text} after using Split ${res3}`);

Reverse & Join

It reverse the string & It converts array to string.

       let text2 = "ShaileshBind";
      var res4 = text2.split("").reverse().join("");
      console.log(res4);

Trim

It removes unnecessary spaces.

     let text3 = "      Shailesh        ";

     `text =${text3} length =${text3.length}  after using trim : text=${res5} length =${res5.length}`
     );
